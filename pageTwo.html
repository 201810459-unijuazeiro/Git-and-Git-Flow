Blog: O PROGRAMADOR

Conteúdo Inicial: Hello World!


"Olá Mundo" é um famoso programa de computador que imprime "Olá, Mundo!", usualmente seguido de uma
quebra de linha, com algumas variações como inexistência do ponto de exclamação e letras em minúscula,
no dispositivo de saída. É utilizado como um teste ou como um exemplo de código minimalista de uma
linguagem de programação. Um programa de propósito semelhante é o algoritmo de Trabb Pardo-Knuth.
